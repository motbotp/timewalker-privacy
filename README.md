# Privacy Policy Site

This folder contains a simple static privacy policy page for GitHub Pages.

## Files

- `index.html`: the privacy policy page

## Publish on GitHub Pages

1. Create a GitHub repository.
2. Upload `index.html` to the root of that repository.
3. In GitHub, open the repository.
4. Go to `Settings` -> `Pages`.
5. Under `Build and deployment`:
   - set `Source` to `Deploy from a branch`
   - choose the `main` branch
   - choose the `/ (root)` folder
6. Save.
7. GitHub will publish the site at a URL like:
   - `https://<your-github-username>.github.io/<repo-name>/`

## Before publishing

Edit `index.html` and replace:

- `[Your App Name]`
- `[your-email@example.com]`
